The source code is in "webcode" directory

After setting your directory in the command promt upto "webcode", type "bundle exec jekyll serve", to run the blog post on the local host.

Your command should look something like this at the end:

"webcode>bundle exec jekyll serve"

============================================

Steps to setup the environment

1. Create a virutal environment using the below code
virtualenv {ENV_FOLDER_NAME}/

2. Activate the environment
source {ENV_FOLDER_NAME}/bin/activate

3. Go to the codebase folder
cd datascience-master/CodeBase/

4. In terminal, install all the required libraries using requirements.txt
pip install -r requirements.txt

5.To open jupyter-lab, use the command jupyter-lab in Terminal

6. Starting the notebooks wrt. the tasks

HAPPY RUNNING!!
